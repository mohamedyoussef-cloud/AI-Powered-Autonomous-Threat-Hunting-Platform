# Methodology

The project will follow an applied research and engineering methodology:

1. Review existing literature and frameworks related to threat hunting, MITRE ATT&CK, SIEM automation, LLM-based code/query generation, and ML-based alert triage.
2. Design a modular architecture for an autonomous threat hunting platform.
3. Implement a prototype using public datasets and simulated SIEM integrations.
4. Conduct experiments on hypothesis generation, query generation, false-positive suppression, and end-to-end hunt workflow.
5. Evaluate the system using measurable metrics such as time saved, query correctness, precision, recall, F1-score, and false-positive reduction.
